# Lua FCGI

Lua Fast CGI module

## Dependencies

There are 2 dependencies needed to build this project
they are:

 * Lua 5.1 development
 * Fast CGI development

You can run the following command to install dependencies

    $ sudo apt-get install liblua5.1-0-dev libfcgi-dev

## Building

    $ make

## Running Luafcgi

    $ lua-fcgi
